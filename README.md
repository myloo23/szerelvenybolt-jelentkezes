# Szerelvénybolt Kft. tesztfeladat

Ez a projekt a Szerelvénybolt Kft. által küldött frontend és backend tesztfeladat megoldása.

## Fájlok

- `index.html` – az oldal alap HTML szerkezete
- `style.css` – a reszponzív megjelenés és animációk
- `script.js` – a termékek betöltése és megjelenítése
- `products.php` – a termékadatok kiszolgálása JSON formátumban
- `products.json` – az eredeti termékadatok külön JSON fájlban
- `export.php` – XML export a készleten lévő termékekből

## Futtatás

A projekt PHP beépített szerverével futtatható.

php -S localhost:8000
